# رفع المشروع إلى GitHub واستخراج APK

## إنشاء المستودع

1. أنشئ مستودعًا جديدًا في GitHub بدون ملفات ابتدائية.
2. فك ضغط الحزمة وارفع **محتويات** مجلد `SawaedEmployeeWebView` إلى جذر المستودع؛ يجب أن يظهر ملف `settings.gradle` ومجلد `.github` في الصفحة الرئيسية.
3. اجعل الفرع الرئيسي `main` أو `master`.

## بناء APK

البناء يبدأ تلقائيًا بعد رفع الملفات إلى الفرع الرئيسي. ويمكن تشغيله يدويًا من:

1. افتح تبويب **Actions**.
2. اختر **Build Sawaed Employee APK**.
3. اضغط **Run workflow**.
4. بعد نجاح المهمة افتح تشغيل الـWorkflow.
5. من قسم **Artifacts** نزّل `Sawaed-Employee-APK-1.11.0`.
6. فك الملف الذي تم تنزيله لتحصل على `Sawaed-Employee-1.11.0.apk`.

هذه نسخة Debug موقعة تلقائيًا ومناسبة للتثبيت والاختبار الداخلي. للنشر في Google Play أو التوزيع المؤسسي الدائم يجب إنشاء Release APK/AAB بمفتاح توقيع خاص بالشركة وعدم رفع المفتاح أو كلمة مروره إلى المستودع.
